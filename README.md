### Miami Rail

#### Dev Dependencies
1. Xcode
2. Node
3. Yarn

#### Dev Setup
1. Clone this repository
2. Run `yarn install`
3. Build `ios/MiamiRail.xcodeproj`

#### Dev
1. Run `yarn start`
2. Open Simulator
